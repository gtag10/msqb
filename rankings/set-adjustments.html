To ensure the most fair results possible, we adjust sets. This means that we can fairly compare Team A and Team B if Team A plays Set A and Team B plays Set B. We achieve this via links.

Read More

Let's say that we have three teams, Team A, Team B, Team C,  and two sets, Set A, and Set B. Teams A and B played both Set A and Set B, while Team C only played Set B. Let's take the following example results

Set A PPB Set A P/G Set B PPB Set B P/G PPB Difference P/G Difference
Team A 15 5 11 3 4 2
Team B 14 4 8 1 6 3
Team C 9 1


By averaging the Points per Bonus (PPB) and Powers per Game (P/G) differences, we find that the links get 4 less PPB (-4) on Set B compared to Set A, and 3 less P/G (-3) on Set B compared to Set A. Assuming we were using Set A as the baseline, or the set that we compare all sets to, Team C's adjusted PPB would be 14, while the adjusted P/G would be 3.5. The adjusted statistics would then be used in all calculations, meaning that Team C could be compared with against any team that only played Set A. Negs per Game is not adjusted due to high variance.
